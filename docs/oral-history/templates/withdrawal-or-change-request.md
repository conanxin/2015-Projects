# 撤回或修改请求 v## · OH-[YYYY]-NNN

> **状态**：项目内操作模板。**正式签字前**请咨询律师。
> **本请求不要求身份证件。** 由操作员和讲述者使用**事前约定**的暗语
> 或通讯渠道验证身份。

| 字段 | 值 |
|---|---|
| 请求日期 | `[YYYY-MM-DDThh:mm]` |
| 项目编号 | `OH-[YYYY]-NNN` |
| 请求人 | `[讲述者本人 / 讲述者书面授权的家属]` |
| 身份验证方式 | `[事前约定暗语 / 公开通讯 / 项目双方签字的撤回通道]` |
| 验证状态 | `[PASS / FAIL]` |

---

## 1. 请求类型（**只选一项**）

| 类型 | 选择 ☐ |
|---|:---:|
| 撤回：完整下架 | ☐ |
| 撤回：部分下架 | ☐ |
| 修改：替换某些段 | ☐ |
| 修改：删除某些段 | ☐ |
| 修改：调整化名或地点 | ☐ |
| 修改：调整公开形式（如改为只家庭可见） | ☐ |

## 2. 范围（操作员协助逐条填写）

### 2.1 受影响的项目 ID 与文件

```
- 项目: OH-[YYYY]-NNN
- 受影响文件路径（不写进公共 Git）：
  - /site/src/content/stories/<slug>.md
  - /site/src/data/stories/<slug>.segments.json
  - …
```

### 2.2 受影响的具体章节或字段

| # | 类型 | 描述 |
|---|---|---|
| 1 | `[章节 / 字段 / 段落]` | `[具体描述]` |
| 2 | `[类型]` | `[具体描述]` |
| … | … | … |

### 2.3 是否撤回"非可控"副本（搜索引擎、社交媒体等）

| # | 系统 | 是否已知 | 操作 |
|---|:---:|:---:|---|
| 1 | Google / Bing | ☐ | `[noindex / sitemap remove]` |
| 2 | `web.archive.org` | ☐ | `[exclude request]` |
| 3 | 社交媒体 | ☐ | `[请 X 删除转推]` |
| 4 | RSS 订阅器 | ☐ | `[请 X 重新拉取]` |

> 这些**不在**操作员的完全控制范围内；列出它们是为了**文档化**，
> 而非承诺。

## 3. 期望响应时间

| 选项 | 选择 ☐ |
|---|:---:|
| 24 小时内紧急下架（公开层面已构成风险） | ☐ |
| 7 天内处理（默认） | ☐ |
| 14 天内处理 | ☐ |
| 其它（请注明：___） | ☐ |

## 4. 不需要提供的信息

本请求**不需要**以下任何一项：

- 身份证件号码；
- 真实家庭地址（除非为法律程序）；
- 生物识别（指纹、面部扫描）；
- 第三方平台的账号密码。

> 提供。
