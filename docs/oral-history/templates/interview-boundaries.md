# 采访边界 · OH-[YYYY]-NNN

> **状态**：项目内操作模板。**正式签字前**请咨询律师。
> 本文件让讲述者在采访前预先写下"哪些话题可以不谈"、"哪些可以录
> 但不能公开"等。

| 字段 | 值 |
|---|---|
| 同意日期 | `[YYYY-MM-DD]` |
| 讲述者 | `[化名或代号]` |
| 项目编号 | `OH-[YYYY]-NNN` |
| 本表版本 | `v##` |

---

## 1. 完全不谈（默认"不进入采访"）

请列出您**希望完全不进入这次采访**的话题。例如"不再婚的母亲"、
"前任"、"法律责任中的家庭矛盾"等。

```
1. [话题 1]
2. [话题 2]
3. [话题 3]
4. …
```

## 2. 可以谈但不能公开（默认"录了但不进 Level 2 / Level 3"）

请列出您**可以告诉我、但不希望任何对外发布**的话题。

```
1. [话题 1]
2. [话题 2]
3. [话题 3]
4. …
```

## 3. 可以谈，可以公开但只能概括（不引号引用）

请列出您**希望公开、但只用概述、不引用逐字话语**的话题。

```
1. [话题 1]
2. [话题 2]
3. [话题 3]
4. …
```

## 4. 完全公开（默认"可公开页面的可引用"）

请列出您**希望公开、且可作为短摘录或编辑摘要**的话题。

```
1. [话题 1]
2. [话题 2]
3. [话题 3]
4. …
```

## 5. 第三方独立同意（涉及特定家人 / 朋友 / 单位）

请列出您**提到的特定第三方**，以及他们各自的同意状态。

| # | 第三方代号 | 关系 | 是否同意被化名提到 | 是否同意被真实姓名提到 | 同意日期 |
|---|---|---|:---:|:---:|---|
| 1 | `[代号 1]` | `[关系]` | ☐ | ☐ |  |
| 2 | `[代号 2]` | `[关系]` | ☐ | ☐ |  |
| 3 | `[代号 3]` | `[关系]` | ☐ | ☐ |  |

> 任何第三方拒绝的部分都**必须**删除或概括，**不分担**同意风险。

## 6. 任何"采访当日"再说也可以

您可以选择**今天不写**，而是在采访当日 / 之后再告诉我哪些不公开。
这不影响本项目的其余流程。
